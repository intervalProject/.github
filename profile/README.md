# 인터벌 코치 개인정보 처리방침

시행일: 2026-04-24

인터벌 코치는 사용자의 운동 중 실시간 속도와 이동 거리를 계산하기 위해 기기의 위치 정보를 사용합니다.

## 수집 및 처리하는 정보

- 위치 정보: 운동 중 현재 속도와 이동 거리 계산에 사용됩니다.
- 운동 기록: 운동 시작 시간, 운동 시간, 평균 속도, 최고 속도, 거리, 완료 여부, 목표 속도 미달 횟수가 기기 안에 저장됩니다.
- 사용자 설정 및 프리셋: 인터벌 시간, 세트 수, 목표 속도, 알림 설정이 기기 안에 저장됩니다.

## 사용 목적

위치 정보는 실시간 속도 표시, 거리 계산, 목표 속도 미달 알림, 운동 기록 저장에만 사용됩니다.

## 외부 전송 및 공유

인터벌 코치는 위치 정보, 운동 기록, 사용자 설정을 자체 서버로 전송하지 않습니다. 또한 해당 정보를 제3자에게 판매하거나 공유하지 않습니다.

## 보관 및 삭제

운동 기록은 사용자가 앱의 기록 화면에서 삭제할 수 있습니다. 앱을 삭제하거나 앱 데이터를 지우면 기기에 저장된 운동 기록, 사용자 설정, 프리셋도 삭제됩니다.
